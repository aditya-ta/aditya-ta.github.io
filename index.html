Projects

Audio Beam (Senior Year Design Project)

This product is a portable guitar effects processor that uses inbuilt DSP functionalities in the MSP430 MCU. I lead the software development in a team of four that sucessfully delivered this project. It aimed to make the process of using an electric guitar more convenient. The device is powered using a lithium ion battery and has the ability to be recharged. The guitar can be hooked to the device via a traditional quater inch jack and controlled wirelessly through an android phone using an application that we specifically made for this device. This device has equalization, reverb and distortion functionalities. I developed and tested embedded software for the MCU-Codec interfacing using SPI and DMA (a work around for I2S that our MCU lacked). I interfaced the BLE(Bluetooth Low energy) module(RN4020) to the MCU using UART and this enabled us to change the sound settings on the device. Additionally, I collaborated in the testing and finetuning of digital filters.
